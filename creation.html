レトロデザイン制作の過程

AeroやレトロWebデザインを現代的に再現する

Tailwind CSS

ユーティリティクラス（小さな機能ごとのクラス）を組み合わせて、素早く美しいWebデザインを作れるCSSフレームワーク。
複雑なCSSを書かずに、HTMLのクラス名だけでデザインを調整できるのが特徴です。

Aeroデザイン

Windows Vista/7時代に登場した「ガラスのような半透明・ぼかし・光沢感」が特徴のUIデザイン。
奥行きや透明感、グラデーションを多用し、立体的で未来感のある見た目を演出します。

Aquaデザイン

AppleのMac OS Xで採用された「水滴のような透明感・光沢・丸み」が特徴のUIデザイン。
カラフルで柔らかい印象、ガラス玉やジェルのような質感がポイントです。

アイデア発想

まず、Wayback Machineなどで2000年代のWebデザインを調査し、
AeroやAquaなどの特徴をリストアップ。
どんな要素が「レトロ感」や「未来感」を生み出しているのかを分析し
以下の要素を特定しました。

  • グラデーションや半透明、ガラス風のパネル
  • 大きなロゴや立体的なボタン
    • 立体的なボタン：影やグラデーションで浮き上がって見えるボタン。押しやすさ・目立たせるために使います。
  • アイコンやイラストの多用
  • 青系・水色系の配色
    • 配色：色の組み合わせ。青系は信頼感や清潔感を与えるため、レトロデザインでもよく使われます。

デザイン設計

Tailwind CSSを使い、Aero風のガラス感やグラデーション、立体感を再現。
レスポンシブ対応や、ボタン・カードのホバー演出も工夫しました。

  • ぼかし, 透明度, 影, ふちの活用
    • ぼかし：背景やパネルの後ろを少しにじませて、奥行きやガラス感を出す効果。
      影：要素の下や周りに影をつけて立体感を出すテクニック。
      ふち：枠線や縁取り。要素を目立たせたり、区切りをつけるために使います。
  • 画像を入れてレトロデザインを強調
  • アニメーションやグロー効果
    • アニメーション：動きをつけて注目させたり、操作感を向上させる演出。
      グロー効果：光っているような見た目。ボタンやアイコンのホバー時に使うと目立ちます。
  • セクションごとに異なるグラデーション
    • グラデーション：セクションごとに色の流れを変えることで、ページ全体にリズムや変化をつけます。
